# 2.5D Action RPG

一個使用 Godot 4.4 開發的 2.5D 動作角色扮演遊戲。

## 遊戲特色

### 場景一：遊戲模式選擇
- 1 Player（單人遊戲）
- 2 Players（雙人遊戲）
- 1 Player + AI（單人 + AI 協作）

### 場景二：職業選擇
1. **戰士** - 近戰為主，威力強大
2. **魔法師** - 冰火雷法術，HP 較少，依賴 MP 瓶，移動速度較慢
3. **獵人** - 弓箭攻擊，可蓄力造成更強傷害，技能冷卻時間較長
4. **牧師** - 使用 MP，擁有回復能力和神聖攻擊法術

### 場景三：遊戲關卡
- 共 3 個關卡
- 每關有多個敵人
- 隨機寶箱和敵人掉落物品
- 物品類型：
  - HP 補充（小）
  - HP 補充（中）
  - MP 補充（小）
  - MP 補充（中）

## 操作方式

### 玩家 1
- W/A/S/D - 移動

### 玩家 2
- 方向鍵 - 移動

## 開發狀態

基礎架構已完成，包含：
- 主選單系統
- 職業選擇系統
- 遊戲管理器
- 玩家控制（支援雙人和 AI）
- 敵人 AI
- 物品掉落系統
- 寶箱系統
- 關卡進度管理
